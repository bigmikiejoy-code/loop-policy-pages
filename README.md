# Loop Policy Site

This repository is intentionally minimal and public.
It hosts only compliance pages for app-store review.

## Included pages

- /account-deletion/
- /child-safety/
- /privacy/
- /terms/

## Deploy

1. Push this repository to GitHub.
2. In repository settings, enable Pages with source set to GitHub Actions.
3. Run the workflow Deploy Policy Pages.

## Live URL pattern

https://YOUR_USERNAME.github.io/YOUR_POLICY_REPO/
